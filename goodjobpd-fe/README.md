# goodjobpd-fe (v3)

칭찬 포도알 레이스 프론트엔드 (Vite + React)

## 이번 변경 사항

1. 포도송이 역삼각형 배치
   - GrapeCluster 컴포넌트에서 7-6-5-4-3-3-2 패턴으로 포도알 배치
   - 각 포도알은 더 크고(40px) 보라색 그라디언트 + 그림자 적용
   - hover 시 툴팁으로 상세 정보 (유형, 시간, 메모, 사진 여부) 표시

2. 경주별 하루 포도알 제한
   - 경주 생성 시 `하루 최대 포도알 수` 설정 (기본 2)
   - 경주 목록 / 경주 상세 / 포도 등록 화면에서 해당 값 표시
   - 실제 제한 로직은 백엔드에서 검증 (하루 N개 초과 시 에러 메시지 반환)

3. 포도알 활동 일시 입력
   - 포도알 등록 화면에서 `datetime-local` 입력 지원
   - "지금 시간" 버튼으로 현재 로컬 시간을 바로 입력
   - 입력한 시간이 그대로 백엔드의 `activityTime`으로 전달되어, 일별 제한 및 화면 표시 기준이 됨

4. FINISHED 경주에 포도 등록 금지
   - 경주 상세 화면: 상태가 FINISHED인 경우 포도 등록 버튼 disabled + 안내 문구 표시
   - 포도 등록 화면: FINISHED 상태일 때 전체 폼이 disabled + 안내 문구
   - 백엔드에서도 FINISHED 경주에 대해 포도 등록 시 에러 처리

## 기본 실행 방법

```bash
npm install
npm run dev
```

- 기본 포트: http://localhost:3000
- 백엔드: http://localhost:8080 (goodjobpd-api)
